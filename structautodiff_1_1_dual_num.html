Detailed Description

Represents the "real" part of a number in member r and the derivative in d.

Can be converted to and from doubles as required, so that generic code such as: 

 template <class NumT>
 NumT doIt(NumT x) {
     NumT y = x;
     for (int i = 0; i < 100; i++) {
         y = dSin(x + y);
     }
     return y;
 }

will work for both doubles and DualNums.
